I would like to share with you some of the projects that I've worked on in the past, and in the present day. I've included some basic information about the sites as well, for reference.

More projects will be listed soon, as I get time to write these up!

Open Run Log Open Run Log is a run tracker built using Tornado, Redis, and Mongo. It also includes some D3 javascript elements as well as MapReduce (in Mongo). It was stared by my friend David, and is open source, available at GitHub.
The Button (WebSockets) (deprecated) I've been playing with some WebSockets, and this is where I've been doing my work. The repository for this code is at GitHub.
ReminderCall.me (deprecated) This is a tool built with Twilio to schedule reminder calls/texts. It is also built using Flask, RabbitMQ, and MySQL.
University of Michigan Men's Glee Club While I was the Glee Club's webmaster, I rewrote the website from scratch using the PHP framework CodeIgniter, and MySQL.
